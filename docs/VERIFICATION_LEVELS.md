# Verification levels

## Verified calculation

A formula or cost curve is represented directly, has source/version metadata, and passes tests.

## Reference-aligned

A native result has been compared with an imported trusted reference and agrees within documented tolerance.

## Estimated calculation

A partial model uses explicit assumptions. The interface must state those assumptions.

## Strategic heuristic

A relative priority based on account context, opportunity cost, or common strategy. It is not an exact ROI value.

Contributors must not upgrade a confidence label without evidence and tests.
